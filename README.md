### cout语句的使用方法
先来看最基础的输出——字符串输出
示例：输出Hello World！
``` cpp
cout << "Hello World!";
```
没错就这么简单
只需要写上两个小于号里面加上双引号就可实现字符串输出
现在来看下一个类型的cout——字符
``` cpp
cout << 'a';
```
这里你可能会问：字符是什么
字符类型用于存储 ASCII 字符，通常占用 1 个字节。
现在来看最后一种输出 输出一个数值
``` cpp
cout << 1 + 1; // 可以写一个算式
cout << 5; // 也可以是直接的一个数字 包括小数 负数
cout << ((2 + 3) * 678) / 3; // 复杂的算式也可以
```
